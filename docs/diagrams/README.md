# Diagrams

This folder contains architecture and flow diagrams for the project.

- `project_architecture.mmd`: Mermaid source for high-level module architecture.
- `project_architecture.svg`: SVG image version of the high-level architecture.
- `runtime_flow.mmd`: Mermaid source for the request execution flow.
- `runtime_flow.svg`: SVG image version of the runtime flow.
- `report_system_methodology.mmd`: report-focused architecture + methodology flow (with decisions and retry loop).

Open `.mmd` files in Mermaid-compatible viewers, or open `.svg` files directly in any browser.
